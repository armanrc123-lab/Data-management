DESCRIPTION
	The purpose of Team 71 Final Project is to take a list of dictionaries representing students and their information and either assist in analyzing it or adding to the list of students by calculating and adding a grade average for example. It can take the list of students as input and it can output that list organized as a graph and it can organize the list in ascending/descending order.


DEPENDENCIES
- Python 3.13.0
- Matplotlib 3.9.3
- NumPy 2.1.3
- Counter 1.0.0


INSTALLATION
To run software, first install the required dependencies and open Python. Then, open the program "Team 71 Final Project". Make sure you have your list of dictionaries on hand. Once the file is open, you can call any of the functions and use your list as an input.
